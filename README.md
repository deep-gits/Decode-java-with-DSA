Welcome to the **Decode Java with DSA** repository! This repository serves as a comprehensive collection of Data Structures and Algorithms implemented in Java. It is designed to track my learning journey, store efficient solutions to algorithmic problems, and serve as a quick reference guide.

## 📌 About This Repository

This repository contains:
* **Concept Implementations:** Core data structures coded from scratch.
* **Problem Solving:** Solutions to standard problems from LeetCode, HackerRank, GeeksforGeeks, etc.
* **Optimized Code:** Focus on Time and Space Complexity ($O(N)$, $O(\log N)$, etc.).
* **Course Notes:** Code written while following the "Decode Java with DSA" curriculum.

## 📚 Topics Covered

Here is the roadmap of topics covered in this repository:

### Phase 1: Basics & Fundamentals
- [x] Variables, Data Types & Operators
- [x] Control Flow (If-Else, Switch, Loops)
- [x] Methods & Functions
- [x] Time & Space Complexity Analysis

### Phase 2: Core Data Structures
- [ ] Arrays (1D & 2D)
- [ ] Strings & StringBuilder
- [ ] Searching (Linear & Binary Search)
- [ ] Sorting (Bubble, Selection, Insertion, Merge, Quick)

### Phase 3: Advanced Data Structures
- [ ] Linked Lists (Singly, Doubly, Circular)
- [ ] Stacks & Queues
- [ ] Trees (Binary Trees, BST)
- [ ] Heaps & Priority Queues
- [ ] HashMaps & HashSets

### Phase 4: Algorithmic Paradigms
- [ ] Recursion & Backtracking
- [ ] Greedy Algorithms
- [ ] Dynamic Programming (DP)
- [ ] Graphs (BFS, DFS, Shortest Paths)
